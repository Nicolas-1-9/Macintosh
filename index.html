Estructura General

la cpu creada por Zenon Kuc y Sandy Hiser en los años noventa implementan la arquitectura RISC (Reduced Instruction Set Computer) dejando de lado la CISC (Complex Instruction Set Computer) para mejorar la eficiencia de diseño y uso

Procesador (CPU)

El procesador es el cerebro de la computadora, encargado de ejecutar instrucciones y procesar datos, el procesador en colaboracion entre Apple, IBM y Motorola fue de los primeros usados en la linea de consumidores profesionales de apple como ingenieros y diseñadores, con velocidades de 66/80/100/120 MHz, 2.8 millones de transistores, tamaño de bus de datos de 32 bits y hasta 3 instrucciones por ciclo

Random Access Memory (RAM)

las dos tarjetas ram tienen chips etiquetados con la serie d421256 SIL8V-12JP y 3M 0957-4 lo cual indica que una es de 8 mb y la otra de 4 mb de marca Fujitsu

Placa Base Motherboard

la placa base cuenta con diferentes ranuras como lo son, SIMM de 72 pines, puerto DB-15 para el monitor o conocidos como ADB (Apple Desktop Bus) tambien usados para perifericos, ranuras de expansion llamadas NuBus lo que permite añadir tarjetas de video o tarjetas de red adicionales.

Dram Simms

DRAM: Dynamic Random Access Memory (Memoria de Acceso Aleatorio Dinámico) llamada dinamica porque necesita ser refrescada cada cierto tiempo para mantener la informacion, mas barata y mas eficiente que la static ram, tambien usada en la epoca SIMM: Single In-line Memory Module (Módulo de Memoria en Línea Simple) significa simplemente el metodo la encapsular la informacion mediante contactos electricos y pines, que van desde 30 hasta 90 pines

Tarjeta grafica

la video random access memory es un tipo de memoria diseñada específicamente para almacenar datos gráficos, e utiliza para almacenar imágenes, texturas, y otros datos necesarios para renderizar gráficos en la pantalla

Disipador

el disipador es usado como medio de enfriamiento para el procesador, en esta epoca se usaba simplemente pasta termica y una estructura metalica porque en la epoca no habria muchos programas para generar demasiado estres a el procesador

Lector de Discos Compactos

el lector de discos compactos permite leer informacion almacenada mediante un lector laser y un fotodetector, asi como permite tambien insertar o "quemar" informacion dentro

Fuente de Poder

la fuente de poder es usada para alimentar a toda el harware del computador, mediante un transformador, rectificadores de voltaje (convertir corriente alterna a corriente continua) y reguladores de voltaje, y , antes no usado pero implemetado años despues, los breackers, dispositivos que interrumpen la corriente electrica para prevenir los daños de los otros componentes en caso de una anomalia

Disco Duro

cuenta con un disco duro (unico en la epoca,precursor de los NVDme o SSD (disco de estado solido)) que funciona mediante platos, que son discos magneticos que giran a gran velocidad generando asi la posibilidad de almacenar y leer informacion, en esta epoca los almacenamientos oscilaban entre 100 mb y algunos gb, lo cual actualmente seria contraproducente

Parlante

el parlante conectado directamente a la motherboard usando una bobina movil e imanes que permitian transformar electricidad en ondas sonoras

Lector de Disckettes

el disckette junto a su lector era la manera mas eficiente de almacenar datos e archivos, funcionaba con un disco metalico muy parecido a el disco duro, y en la parte de arriba tiene el cabezal de lectura e escritura, usado para reescribir los datos del disco flexible
